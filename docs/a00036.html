bool scene_Decode ( SCENE *  scene,
const char *  filename 
)

Load a scene from a file. If this function succeeds you must destroy the SCENE with scene_Destroy. If this function fails all memory usage is cleaned up and you will not need to use scene_Destroy.

Parameters
sceneAn uninitialized SCENE to read data into
filenameThe file to read data from
Returns
Whether the decode worked.
