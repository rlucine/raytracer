Detailed Description

Generalized storage of shape data. The SHAPE_TYPE must correspond exactly to the target of the data pointer. SHAPE_SPHERE corresponds to a SPHERE. SHAPE_ELLIPSOID corresponds to an ELLIPSOID. No further shape types are supported.
